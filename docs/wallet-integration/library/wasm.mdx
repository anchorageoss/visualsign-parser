VisualSign can be compiled to WebAssembly for use in browsers, Node.js, and other JavaScript environments.

## Building for WASM

```bash
# Install wasm-pack if needed
cargo install wasm-pack

# Build the WASM package
wasm-pack build --target web
```

## Browser usage

```html
<script type="module">
import init, { parse_transaction } from './pkg/visualsign_parser.js';

async function parseEthereumTx(rawTx) {
    await init();
    return parse_transaction(rawTx, 'ethereum', null);
}
</script>
```
